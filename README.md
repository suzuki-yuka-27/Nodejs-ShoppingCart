# Nodejs-ShoppingCart

## 技術スタック
| カテゴリ    | 技術                                 |
| ------- | ---------------------------------- |
| フロントエンド | HTML5、CSS3、JavaScript（ES6+）、jQuery |
| バックエンド  | Node.js、Express                    |
| データ形式   | JSON                               |
| データ保存   | JSONファイル（`fs`モジュール）                |
| 通信      | Ajax（`jQuery.ajax()`）              |
| CSV出力   | `csv-writer`                       |
| テスト     | Jest                               |
| パッケージ管理 | npm                                |
| 開発環境    | Visual Studio Code                 |
| バージョン管理 | Git、GitHub                         |
